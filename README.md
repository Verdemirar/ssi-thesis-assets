# ssi-thesis-assets

Public, image-only chart assets for ssi-thesis theses, served via jsDelivr CDN.

**Only image files live here.** No thesis text, no data, no source. Pushed automatically by `scripts/publish_charts.py` in the (private) ssi-thesis repo.

CDN base: `https://cdn.jsdelivr.net/gh/Verdemirar/ssi-thesis-assets@main/charts/`
